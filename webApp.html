                              Store Management System

                              January-May 2020


                              WHAT?


                              I designed and coded a store management system for a fictional creamery. This application can help admistrators manage their business by keeping track of payrolls, employees, and stores.

                              WHY?


                              This project was part of a semester course I took titled: Application Design and Development. This course taught many of the core fundementalls of application development including: authentication & authorization, frontend developement, and MVC principles. The term project for this course requred the full-implementation of a store management system.

                              HOW?


                              For this project, I worked individually primarity with the programming tools Ruby and rails. I also worked with sqlite and bootsrap as well. To learn more about this project checkout the Github repository!
