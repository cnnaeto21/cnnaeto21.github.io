                              67-240: Braddock Region Chamber of Commerce Website

                              December 2020


                              WHAT?


                              A website design prototype for the Braddock Region Chamber of Commerce (BRCC) in the Pittsburgh area. The BRCC is a new community center thatwants to act as a resource center for the Braddock region. Myself and two of my classmates worked with them to create a prototype for what they envisioned their online presence to be.

                              WHY?


                              Creating this website was part of the curiculum for a course I took titled: Mobile Web Design & Development (67-240). This course provided an introduction to the foundations of Web Design as well as some of the relavent tools and resources necessary to be a succesful designer. For our final project we implemented and hosted a site for the BRCC.

                              HOW?


                              For this project, I worked in a group of three students as we developed wireframes, user profiles, and conducted user research all in which contributed to our final prototype. Some of the tools and languages we used are:

                              • Figma
                              • HTML/CSS
                              • Javascript
                              • Github
                              • Bootstrap
