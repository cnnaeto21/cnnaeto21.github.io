                              E-commerce platform (Udemy Course)

                              May - August 2020


                              WHAT?


                              I built a general E-commerce platform. This site could be used to purchase and sell physical and digital products.

                              WHY?


                              I built this site as a result of udemy couse I completed over the couse of one summer. The goal of this app was to learn the step by step process of building and launching an E-cocommerce platform. Learn more about the course here.

                              HOW?


                              This project was built primary in the python based web framework Django. I was also exposed to a number of useful API's includeing Stripe and Mailchimp. Lastly I learned some of the requirements of web hosting by using Heroku and AWS.
